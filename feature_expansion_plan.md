# WeatherBot 2.0 功能扩展计划

本计划旨在利用彩云天气和和风天气 API 的全部潜力，通过高级特性“赋能” WeatherBot。

## 1. 视觉增强 (图表功能)

利用 `matplotlib` 生成美观、易于分享的图表。

### 1.1 温度趋势图

- **目标**: 可视化展示气温随时间的变化趋势。
- **数据源**:
  - 逐小时预报 (48 小时/24 小时) - 来自彩云/和风。
  - 逐日预报 (15 天/7 天) - 来自彩云/和风。
- **实施方案**:
  - 在 `Visualizer` 类中添加 `draw_hourly_temp_chart` 方法。
  - 在 `Visualizer` 类中添加 `draw_daily_temp_chart` 方法。
  - 风格: 深色模式 (Dark Mode)，平滑曲线，清晰的标签。

### 1.2 增强型降雨雷达

- **目标**: 改进现有的降雨图表。
- **数据源**: 分钟级降水数据 (彩云)。
- **实施方案**: 使用渐变色或清晰的颜色编码 (小雨/中雨/大雨) 来增强可读性。

## 2. 主动智能 (推送通知)

扩展 `scheduler.py`，不再局限于降雨提醒。

### 2.1 每日早安简报 (早安天气)

- **目标**: 每天早晨 (例如 7:00 或 8:00) 发送天气摘要。
- **内容**:
  - 今日最高/最低温。
  - 天气状况摘要。
  - 穿衣建议 (基于生活指数)。
  - 降雨概率。
- **逻辑**:
  - 遍历所有订阅的用户/群组。
  - 发送一张精心排版的“每日天气卡片”。

### 2.2 灾害天气预警广播

- **目标**: 即时通知危险天气 (台风、暴雪、暴雨)。
- **逻辑**:
  - 定期检查 `weather.alerts` 字段。
  - 过滤高严重等级 (橙色/红色) 的预警。
  - 推送格式: "⚠️ **[红色] 台风预警** 上海发布详细内容..."

## 3. 专业模式

添加新的斜杠命令或交互按钮。

### 3.1 天文视图 (`/tq astro`)

- **目标**: 服务摄影师和天文爱好者。
- **数据**: 日出日落、月出月落、月相 (和风/彩云)。
- **格式**: 美观的列表或简单的 ASCII 图形。

### 3.2 生活指南视图 (`/tq life`)

- **目标**: 专注于“舒适度”和生活建议。
- **数据**: 紫外线指数、湿度、穿衣指数、流感风险、洗车指数。
- **格式**: 分组列表 (目前已部分在 `indices` 视图中实现，需使其更易访问)。

## 4. 实施步骤

1.  **更新 `Visualizer`**: 添加绘制温度图表的方法。
2.  **更新 `Scheduler`**: 添加早安任务和预警任务。
3.  **更新 `Bot`**: 为新的视觉指令添加处理器 (例如“查看图表”按钮)。
4.  **优化 `Formatter`**: 为简报和预警创建专用模版。

## 5. 时间表与优先级

- **P0**: 视觉图表 (温度) - _高“Wow”因素 (令人眼前一亮)。_
- **P1**: 早安简报 - _高用户留存价值。_
- **P2**: 灾害天气预警 - _安全价值。_
- **P3**: 天文模式 - _小众价值。_
